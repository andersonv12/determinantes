# Determinantes
Sistemas web para cálculo de determinantes e resolução de sistemas lineares

Estes sistemas foram desenvolvidos pelos alunos do 2º ano do curso técnico em informática integrado ao ensino médio do IFF campus Bom Jesus, como resultado de uma atividade proposta de integração entre as disciplinas de Matemática e Desenvolvimento Web (Tecnologias para Desenvolvimento de Software I).
